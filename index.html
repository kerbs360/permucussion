Click HERE to download two Java applications created for drummers/percussionists - perMUmet is a basic general-purpose metronome - perMUbeat is a beat generator that once a beat is entered you can experiment with permutations of that beat. BIG NOTE - These applications were created and implemented in 2010 - before iOS and Android "apps" took off. They were created in Java and should run in any enviornment that the Java Runtime Environment (JRE) has been installed. If you do not know, the .jar extension is similar to the .zip format you might be familiar with for windows compressed file archives. The applications are not installed on your computing device. If they do not run on your device, most likely, the JRE probably is not installed on your device. In the future, we might create updated versions optimized for current hardward. Until then, Have Fun!
